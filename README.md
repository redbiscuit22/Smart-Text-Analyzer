## Project Overview  

Smart Text Analyzer is a PHP-based web application that demonstrates system integration principles through machine learning library integration.  
The application analyzes text content to provide the following:

- **Sentiment Analysis:** Classifies text as positive, negative, or neutral  
- **Keyword Extraction:** Identifies important keywords using TF-IDF algorithm  
- **Emotion Detection:** Detects seven primary emotions in text  
- **Text Summarization:** Creates concise summaries of longer texts  
- **Modern Interface:** Clean, responsive design with dark/light mode
